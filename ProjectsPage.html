New Customer Form

This was one of the projects completed during my technology internship with a regional accounting firm. The goal was to redesign an existing form used by internal staff to add new clients to various internal company databases and resources. By completing this application multiple teams were able to save countless hours, reduce errors, and streamline processes. In addition to a new form I also implemented home pages for staff to reference work and track progress. The technologies used include .Net Core, Blazor, Microsoft SQL, and several other resources.

Through this experience I was able to leverage my software, communication, and organization skills to help create a valuable resource for the organization. Additionally, I learned a great deal about the importance of project planning and how critical management skills are to the overall process and the time it takes to deliver a product. Teamwork is also very important especially in the context of a project involving multiple business teams.

Address Book

This was a class assignment designed to work as an electronic address book taking in various pieces of information about contacts. It offers the ability to use a number of multi-level sorting functions, to add and remove contacts, and an option to display single contacts based on an e-mail address. The program was made in Python using PyCharm.

The GitHub repository can be found here: Address Book.
